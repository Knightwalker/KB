Sorting 2D Array Alphabetically

Functions we apply:


  • usort

    - sort an array by values using a user-defined comparison function

  • strcmp

    - case-sensitive comparison between 2 strings

  • strcasecmp

    - case-insensitive comparison between 2 strings

Sorting modes:


  • strcmp(array $a, array $b)

    - sort elements (Order by value => ASC)

  • strcmp(array $b, array $a)

    - sort elements (Order by value => DESC)

1.

usort

uses an implementation of the Quicksort algorithm to sort the array, it calls your custom

cmp

function as many times as it needs to fully sort the array. For each comparison it calls your callback with two elements and then decides if it needs to swap them or not.

3. Since we have given

usort

an Array of

Arrays

, the function will sort the

Arrays

themselfs and in order for this to happen, we need to specify in which

key

it needs to look for the value of comparison, otherwise we will get the following error

strcmp() expects parameter 1 to be string, array given.

Usually as said in the official PHP documentation documentation

usort

should always reference to the first index of the array, however in some situations this will not happen.

2.

strcmp

will compare 2 strings and return the following:
0: if str1 and str2 are equal
-1 (<0): if str1 is less than str2
1 (>0): if str1 is greater than str2


4. Finally the sort will happen according to the following logic
If

usort

receives 0 then $a is the same priority as $b, keeping the same in the array
If

usort

receives -1 then $a is higher priority than b $b, moving $b down in the array
If

usort

receives 1 then $b is higher priority than a $a, moving $b up in the array
