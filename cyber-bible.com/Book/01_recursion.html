Recursion

In computer science recursion is a problem solving technique, which divides a problem into subproblems of the same type.

  • Recursion happens when a method or a function calls itself.
  • The recursive call can be direct (the method calls itself directly) or indirect (through other methods).
  • The method should have a base case (exit criteria, bottom). Each step on the recursion must move towards the base case.

1. Recursive Factorial

In mathematics, the factorial of a positive integer n, denoted by n!, is the product of all positive integers less than or equal to n.

Base Case: The value of 0! is 1.

Example:
5! = 5 * 4! = 5 * 4 * 3 * 2 * 1 * 1 = 120
4! = 4 * 3! = 4 * 3 * 2 * 1 * 1 = 24
3! = 3 * 2! = 3 * 2 * 1 * 1 = 6
2! = 2 * 1! = 2 * 1 * 1 = 2
1! = 1 * 0! = 1 * 1 = 2
0! = 1
Input
Output
5
120
static void Main(string[] args) { int n = int.Parse(Console.ReadLine()); int factorial = Factorial(n); Console.WriteLine($"{n}! = {factorial}"); } static int Factorial(int n) { if (n == 0) { return 1; } return n * Factorial(n - 1); }

2. Recursive Array Sum

static void Main(string[] args) { var numbers = new[] { 1, 2, 3, 4, 5 }; var sum = Sum(numbers, 0); Console.WriteLine(sum); } static int Sum(int[] array, int index) { if (index == array.Length) { return 0; } var currentSum = array[index] + Sum(array, index + 1); return currentSum; }

3. Recursive Drawing

Recursive methods have 3 parts:

  • Pre-actions (before calling the recursion)
  • Recursive calls (step-in)
  • Post-actions (after returning from recursion)

using System; class Program { static void Main(string[] args) { int n = 5; RecursiveDrawing(n); } static void RecursiveDrawing(int n) { if (n == 0) { return; } // Pre-Action: print n asterisks Console.WriteLine(new string('*', n)); // Recursive call: print figure of size n-1 RecursiveDrawing(n - 1); // Post-Action: print n hashtags Console.WriteLine(new string('#', n)); } }
